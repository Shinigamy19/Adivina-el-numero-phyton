# 🎯 Juego: Adivina el Número

Este es un juego simple en Python donde el jugador debe adivinar un número secreto generado aleatoriamente. El jugador tiene 5 intentos para acertar y recibe pistas después de cada intento.

## 🚀 Cómo jugar

1. El programa genera un número aleatorio entre 0 y 100.
2. El jugador tiene 5 intentos para adivinar el número.
3. Después de cada intento, el juego indica:
   - Si el número ingresado es mayor o menor que el número secreto.
   - El nuevo rango entre el mínimo y el máximo posibles.
4. Si el jugador adivina correctamente, gana.
5. Si se quedan sin intentos, el juego termina con un mensaje indicando cuál era el número secreto.

## 🧠 Lógica del juego

- Se usa `random.randint(0, 100)` para generar el número secreto.
- Se controla el número de intentos con una variable.
- Se actualiza el rango después de cada intento incorrecto.
- Se valida la entrada del usuario para que solo se acepten números enteros dentro del rango.

## 📦 Requisitos

Este juego está escrito en **Python 3**. No requiere librerías externas, solo:

```python
import random
```

## ▶️ Cómo ejecutar

```python
python adivina_el_numero.py
```
